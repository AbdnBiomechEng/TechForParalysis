Getting started

Windows

  1. Install FTDI-Driver
  2. Connect RehaMove3 via USB
  3. Change the latency standard settings
    • Open the device manager
    • Select the virtual COM port in ports and open the properties window
    • Navigate to the port settings, then open the advanced window
    • Set the latency timer to 1 ms (default 16 ms)
  4. Include the headers and the library for your compiler and operation system
  5. Write your application (Using the library)

Linux

  1. Ensure that the kernel includes support for the FTDI-chip
  2. Connect RehaMove3 via USB
  3. Adjust the FTDI latency timer to 1 ms (see e.g. https://projectgus.com/2011/10/notes-on-ftdi-latency-with-arduino/)
  4. Include the headers and the library for your compiler and operation system
  5. Write your application (Using the library)
