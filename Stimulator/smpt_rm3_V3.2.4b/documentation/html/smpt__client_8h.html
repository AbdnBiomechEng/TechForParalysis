SMPT_API bool smpt_new_packet_received ( Smpt_device *const  device)

Checks if one or more packets have been received since the last call.

This function triggers the read of the serial device. The packets are buffered internally. If a new packet (ack) is received, use smpt_last_ack() to get more information about the packet.

Parameters
[in,out]deviceThe device struct is needed for internal handling.
Returns
