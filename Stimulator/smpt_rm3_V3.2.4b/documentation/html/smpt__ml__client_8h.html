SMPT_API bool smpt_send_ml_get_current_data ( Smpt_device *const  device,
const Smpt_ml_get_current_data *const  ml_get_current_data 
)

Builds the Ml_get_current_data packet and sends it to the serial port.

This command is used as a keep-alive signal. After the start of the stimulation (Ml_update) you need to send this command at least every 2 seconds. The response contains general and requested status information.
