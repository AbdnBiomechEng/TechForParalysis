RehaMove3

Keypad

Buttons

  • Power button
    • Push the button to switch on the RehaMove3.
    • The device can be switched off by push and hold the button for at least 5 seconds.
    • After the device has been switched off, a 5 seconds break is required before the RehaMove3 can be switched on again.
  • Minus, plus, function button
    • These buttons have no function currently.

LEDs

  • Power-LED (green/red/blue)
    Color LED state Description
    None Off The device is switched off.
    Blue On The power supply is connected. The device is switched off.
    - - -
    Green On The device is switched on. The rechargeable battery is fully charged.
    Green Fast blinking The rechargeable battery is charging (the power supply is connected).
    - - -
    Green Slow blinking The rechargeable battery state is normal (21-100%).
    OrangeSlow blinking Low rechargeable battery (11-20%); Please connect power supply.
    Red Slow blinking Very low rechargeable battery (<11%); Please connect power supply.
    - - -
    OrangeOn The device is switched on. The device has no rechargeable battery.
  • Stim-LED (yellow/red)
    Color LED state Description
    YellowOn, Off, blinking The LED is switched on during the stimulation (at least one channel is active).
    Red Fast blinking Electrode error at least on one channel
